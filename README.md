# Onokai

Theme for ones who love One Monokai so much!

# How to use

- Clone theme

```shell
git clone git@github.com:cpea2506/mdbook-onokai-theme.git
```

- Copy all files from this repo folder to the theme folder in your `mdbook`

- Config book.toml

```shell
[output.html]
theme = "onokai"
default-theme = "onokai"
```

- Enjoy!
